# 발달노트 · 설치 설명서

## 들어 있는 파일
- `index.html` — 발달노트 앱 본체 (그냥 열어도 대부분 기능 작동)
- `manifest.webmanifest`, `sw.js`, `icon-192.png`, `icon-512.png` — '앱처럼' 설치·오프라인용
- `api/refine.js` — Vercel에 올릴 때 쓰는 AI 서버 파일
- `netlify/functions/refine.js`, `netlify.toml` — Netlify에 올릴 때 쓰는 AI 서버 파일
- 앱은 배포한 곳(Vercel/Netlify)을 자동으로 알아채서 작동해요. 둘 중 편한 곳 하나만 쓰면 됩니다.

---

## API 키는 어디서? (중요)
- 키는 **Anthropic**에서 발급받습니다 → `console.anthropic.com` → 로그인 → 결제수단 등록/소액 충전 → **API Keys → Create Key** → `sk-ant-...` 복사.
- Vercel/Netlify는 **앱을 인터넷에 띄우고 그 키를 안전하게 보관**하는 곳이에요. 키를 만드는 곳이 아닙니다.
- 키는 비밀번호와 같아요. `index.html`이나 GitHub에 넣지 말고, 배포 사이트의 **환경변수**에만 넣으세요.
- 비용: 가장 저렴한 모델(Haiku)을 써서 문장 한 번 다듬는 데 보통 몇 원 수준입니다.

## GitHub에는 무엇을 올리나요?
- **zip 파일 자체를 올리는 게 아니라, 압축을 푼 뒤 그 안의 파일들을 올립니다.**
- 올릴 때 `index.html`이 저장소(repository) **맨 위(루트)** 에 오도록 하세요. `api`, `netlify` 폴더도 구조 그대로 함께 올립니다.
- (Vercel만 쓸 거면 `netlify` 폴더와 `netlify.toml`은 없어도 되고, Netlify만 쓸 거면 `api` 폴더는 없어도 됩니다. 그냥 다 올려도 문제 없어요.)

## A. Vercel로 배포하기 (익숙한 경우 추천)
1. 위 파일들을 GitHub 저장소에 올립니다 (`index.html`이 루트에).
2. `vercel.com` 로그인 → **Add New → Project** → 그 GitHub 저장소 선택 → **Deploy**.
3. 배포 후 **Project → Settings → Environment Variables** →
   - Name: `ANTHROPIC_API_KEY`
   - Value: 아까 복사한 `sk-ant-...` 키 → **Save**
4. **Deployments** 탭에서 최신 배포의 **Redeploy** 를 눌러 키를 반영합니다.
5. 생긴 주소(예: `발달노트.vercel.app`)를 열고 '✨ AI로 다듬기'를 눌러 확인.

## B. Netlify로 배포하기
1. 위 파일들을 GitHub 저장소에 올립니다.
2. `netlify.com` 로그인 → **Add new site → Import an existing project** → 저장소 선택 → **Deploy**.
3. **Site configuration → Environment variables → Add a variable** →
   - Key: `ANTHROPIC_API_KEY`, Value: `sk-ant-...` 키
4. **Deploys → Trigger deploy → Deploy site** 로 다시 배포.
5. 생긴 주소를 열고 확인.

## 진짜 '앱처럼' 홈 화면에 설치하기
배포된 주소(https://...)를 휴대폰 브라우저로 연 다음:
- **아이폰(Safari):** 공유 버튼 → **홈 화면에 추가**
- **안드로이드(Chrome):** 오른쪽 위 ⋮ 메뉴 → **홈 화면에 추가** / **앱 설치**

그러면 발달노트 아이콘이 생기고, 전체화면 앱처럼 열립니다. 한 번 연 뒤에는 인터넷이 약해도 기본 기능이 작동해요('AI로 다듬기'만 인터넷 필요).
